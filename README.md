# Malik Tea Stall Website

Static tea stall website ready for deployment on Netlify, Vercel, GitHub Pages, or any simple web host.

## Project Structure

- `index.html` - page structure and business links
- `style.css` - full styling and responsive layout
- `script.js` - mobile menu, smooth scroll, menu tabs, newsletter mailto flow

## Connected Actions

- `Order Now` opens WhatsApp chat
- `Make Reservation` opens WhatsApp with reservation text
- Phone opens dialer
- Email opens mail app
- Address opens Google Maps
- Newsletter opens mail app with a prepared subscription message

- Address text and Google Maps query

## Deployment

1. Keep these files together in one folder.
2. Upload the folder to Netlify, Vercel, or GitHub Pages.
3. Make sure `index.html` stays in the project root.
4. After deploy, open the live URL and test:
   - menu tabs
   - mobile navigation
   - WhatsApp buttons
   - phone/email links
   - social media links

## Notes

- The site uses external CDNs for Google Fonts and AOS animations.
